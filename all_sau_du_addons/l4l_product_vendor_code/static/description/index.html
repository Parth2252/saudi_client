Product Vendor Code

This Plugin helps users to manage specific product code and product name for products. Display the vendor product code and name in the purchase order line and also add/update the product code and name from the purchase order, with this information also shown in reports.



Features

You can add or update the vendor product code and vendor product name from the purchase order.

You can search for products by vendor product code and vendor product name.

Vendor product code and vendor product name are visible in the purchase order and related reports.
