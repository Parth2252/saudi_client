Outgoing Mail Server By User Or Company


Single click to switch mail by user or company.

This module allows to set up outgoing mail servers as per system user or company.

Key Features

Send mail from user email
Send mail from company email
Single click to switch mails by user or company

1. Outgoing mail server configuration based on user

If login user's mail server is configured, then system will automatically start taking that mail server as per the login user.

2. Outgoing mail server configuration based on company

The module gives flexibility to manage sending emails by company if user wants to send email on behalf of company.

(Note: If company email is not configured, and you select by company option, system will start taking admin email user.)

3. Easy email switch from user to company and vise versa

User can switch emails easily from any odoo screen.
